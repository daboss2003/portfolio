# Samson Oluwafemi - Portfolio

A Bloomberg terminal-inspired portfolio showcasing senior UI development expertise, with a focus on fintech solutions.

## Tech Stack

- **React 19** + **TypeScript** - Modern type-safe React
- **Vite** - Lightning fast build tool
- **Tailwind CSS 4** - Utility-first styling
- **Framer Motion** - Smooth animations and micro-interactions
- **React Three Fiber** - 3D particle effects and backgrounds
- **Three.js** - WebGL 3D graphics

## Features

- Terminal boot sequence animation
- 3D particle field background
- Interactive skills dashboard with animated progress bars
- Fintech-themed design (Bloomberg/terminal aesthetic)
- Responsive design for all devices
- Self-taught journey showcase
- Cross-platform development expertise highlight

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Project Structure

```
src/
├── components/
│   ├── layout/      # Navbar, Footer
│   ├── sections/    # Hero, Skills, Experience, Projects, etc.
│   ├── ui/          # Reusable UI components
│   └── three/       # 3D components (particles, scene)
├── data/            # Static data (skills, projects, experience)
└── index.css        # Global styles and Tailwind imports
```

## Design Principles

1. **Terminal Aesthetic**: Dark theme with green accents mimicking financial terminals
2. **Data-Driven**: Skills and stats presented as market data
3. **Interactive**: Heavy use of animations and micro-interactions
4. **Professional**: Clean typography and purposeful design choices
5. **Performance**: Optimized 3D rendering and code splitting

## Deployment

The portfolio can be deployed to any static hosting service:

- Vercel (recommended)
- Netlify
- GitHub Pages
- Cloudflare Pages

```bash
npm run build
# Deploy the `dist` folder
```
